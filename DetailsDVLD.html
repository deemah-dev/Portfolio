04 ‚Äî Local License

Local Driving License Application

The core workflow of the system. An applicant submits a new license application, which then enters a structured three-stage exam process ‚Äî Vision Test, Written Test, and Street Test. Each test can be scheduled, passed, failed, or retaken independently. Only after passing all three tests is the license issued, at which point the person is automatically registered as a driver.

Submit New Local License Application
Schedule, Take & Retake Tests (Applies to All Three Stages)
Issue License After Passing All Tests ‚Äî Person Becomes a Driver
05 ‚Äî International License

International Driving License

Drivers holding a valid local license can apply for an international driving license directly through the system. The process validates the existing local license, collects the required data, and issues the international license ‚Äî with no need to repeat the exam process.

Apply & Issue International License
06 ‚Äî Renewal

Renew Local Driving License

When a license reaches its expiry date, drivers can submit a renewal application through the system. The renewal process verifies the driver's existing record, generates a new license with an updated expiry date, and archives the old one ‚Äî keeping the history intact and fully traceable.

Renew Local License
07 ‚Äî Replacement

Replace a Lost or Damaged License

In cases of a lost or damaged license, drivers can apply for a replacement. The system records the reason for replacement, invalidates the old license, and issues a new one with the same class and expiry.

Replace Local License
08 ‚Äî Detain & Release

Detain & Release Driving Licenses

Authorities can detain an active license directly within the system, marking it as suspended with a recorded reason and date. When the detention period ends or the issue is resolved, the license can be formally released and restored to active status ‚Äî with the full detain/release history preserved.
